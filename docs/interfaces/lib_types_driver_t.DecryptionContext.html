Interface DecryptionContext

DecryptionContext Interface Definition

remarks

The secret key is in BYTES. You can use unicode or other characters in place as long as it adds up to the correct byte length.

param key

The secret key of an absolute byte length, determined by your selected algorithm.

param content

The content to be encrypted with the respective algorithm. This must be converted to a string for sanity sake.

param aad

(OPTIONAL) The Additional Authentication Data to be passed for decryption. This is only provided by the EncryptionResponse if it is used and is handled automagically.

param tag

(OPTIONAL) The Authentication Tag to be passed for decryption. This is only provided by the EncryptionResponse if it is used and is handled automagically.
